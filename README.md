# Braille-E-book
Software for the braille E-book

How to use/implement this code
1. Clone this repo (git clone ...)
2. run cd english2braille-api
3. Create a virtual environment in english2braille-api folder in terminal: python3 -m venv venv
4. Run the virtual environment
    Windows: venv\Scripts\activate
    Mac: source venv/bin/activate
5. Run the requirments.txt file: pip install -r requirements.txt
4. run the input.py file: python3 input.py and insert your word
5. You should then see the output in the correct format (braille and matrix)
